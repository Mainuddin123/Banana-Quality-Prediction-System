# 🍌 Banana Quality Prediction System

## Overview

The Banana Quality Prediction System is an end-to-end Machine Learning application developed to predict banana quality scores using physical, chemical, and environmental characteristics.

This project demonstrates the complete machine learning workflow including data preprocessing, exploratory data analysis, feature engineering, model training, hyperparameter tuning, evaluation, and deployment using Streamlit.

---

## Problem Statement

Banana quality assessment is important for farmers, distributors, retailers, and food quality analysts. Manual quality evaluation can be subjective and time-consuming.

This project uses Machine Learning to predict banana quality scores based on various measurable characteristics, enabling faster and more consistent quality assessment.
